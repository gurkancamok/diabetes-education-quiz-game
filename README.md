## Overview
This project is an interactive, web-based diabetes education tool that uses quiz-based questions and gamification principles to enhance learning and engagement.

Incorrect answers trigger a hangman-style visual feedback mechanism, reinforcing learning through progressive challenge.

## Key Features
- Diabetes education through interactive quizzes
- Gamification with hangman-style feedback
- Lightweight HTML-based implementation
- Designed for patient education and awareness

## Disclaimer
This tool is intended for educational purposes only and does not replace professional medical advice.
